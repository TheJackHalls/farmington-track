# Farmington Track

Astro + TypeScript + Tailwind scaffold for the Farmington Track project.

## Requirements

- Node.js 18+
- npm 9+

## Install dependencies

```bash
npm install
```

## Run the dev server

```bash
npm run dev
```

## Build for production

```bash
npm run build
```

## Preview the production build

```bash
npm run preview
```
